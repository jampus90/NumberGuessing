# Number Guessing Game

A simple console-based number guessing game built with C# and .NET 8.

Inspired by the project idea from [roadmap.sh](https://roadmap.sh/projects/number-guessing-game).

## How to play

1. Choose a difficulty level:
   - **Easy** — 10 chances
   - **Medium** — 5 chances
   - **Hard** — 3 chances

2. The computer picks a random number between 1 and 100.
3. Type your guess. After each attempt you get a hint: too high or too low.
4. Find the number before running out of chances to win!

## Requirements

- [.NET 8 SDK](https://dotnet.microsoft.com/download)

## Running

```bash
dotnet run --project NumberGuessing
```
